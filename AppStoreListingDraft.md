# 大道无形 App Store 上架文案草稿

## 副标题
基金持仓记录与净值追踪

## 描述
大道无形是一款面向个人使用的基金持仓记录工具，帮助你整理自选基金、记录买入卖出、查看确认净值、持仓金额、近期收益趋势和基金历史业绩走势。

你可以通过基金代码添加资产，记录每一笔交易，并按 T-1 已公布净值查看持仓表现。应用支持 iCloud 同步，便于在多台个人设备之间同步资产清单和交易记录。

免费版可录入 5 只基金，交易记录不限制条数。需要添加更多基金或在多台设备间同步时，可以选择月度订阅或一次买断解锁 Pro。

净值与估值数据来自公开网络数据源，仅供个人记录与参考。实际净值请以基金公司公告为准。本应用不提供投资建议、收益预测、交易服务或任何形式的证券投资咨询。

## 审核备注
本应用用于个人基金持仓记录和公开净值展示，不提供基金交易、投资建议、收益承诺或证券咨询服务。用户数据保存在本机，并可通过用户自己的 iCloud 账号进行同步。

测试说明：
1. 打开应用后可使用基金代码添加自选基金。
2. 进入基金详情页可记录买入/卖出交易。
3. 左上角 iCloud 按钮可查看同步状态。iCloud 同步依赖设备已登录 Apple ID 并启用 iCloud。
4. 内购项目：买断 `com.Harold.daodawuxing.lifetime`，月订阅 `com.Harold.daodawuxing.monthly`。进入“数据与同步”面板可查看 Pro 与 iCloud 同步入口。
